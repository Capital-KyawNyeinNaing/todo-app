# Getting started with Create React App

### Node Js Download (LTS version)

### Yarn install globally
```
  npm install --global yarn
```

### Create React App
```
  npx create-react-app my-app
```

##### Use yarn

```
  yarn start
```

##### Use Npm

```
  npm run start
```

### Vscode exxtensions (knn-snippets)
